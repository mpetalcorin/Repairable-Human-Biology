# Datasheet for Dataset

## Dataset name

Repairable Human Biology synthetic molecular dataset

## Motivation

### For what purpose was the dataset created?

This dataset was created to support a computational proof of concept for the hypothesis that biological repairability can be modeled as an integrated molecular state. The dataset is intended to enable exploratory analyses of how ageing-like decline may correspond to the progressive loss of repair access across multiple molecular layers.

### What gap does it address?

Many ageing and chronic-disease studies describe damage, dysfunction, or senescence, but fewer frameworks explicitly formalize whether a biological system remains **repairable**. This dataset was created to test that idea computationally.

## Composition

### What do the instances represent?

Each row represents one **simulated biological sample** with a defined tissue context and ageing-related biological state.

### How many instances are there?

- 2,500 simulated samples

### What does each instance contain?

Each sample contains:

- demographic-like descriptors
  - age_years
  - age_group
  - sex
  - tissue

- molecular features
  - ros_fold_vs_young
  - atp_au
  - etc_activity_pct
  - mtdna_copy_index
  - gammaH2AX_foci
  - IL6_pg_ml_index
  - p16_expr_fold
  - p21_expr_fold
  - proteostasis_pct
  - autophagy_flux_pct
  - telomere_kbp
  - cytosolic_mtDNA_fold

- derived variables
  - repairability_score
  - repair_window_open

### What are the biological states?

The dataset includes four simulated age-related biological states:

- young
- midlife
- older
- aged_stressed

### What are the tissue contexts?

The dataset includes four simplified tissue classes:

- fibroblast
- skeletal_muscle
- hepatocyte_like
- neuronal_like

## Data collection process

### Was the dataset directly observed or measured?

No. The dataset is **synthetic**.

### How was it generated?

The dataset was generated computationally using:

- age-group-specific anchors
- tissue-specific modifiers
- normal and log-normal sampling
- mechanistic dependency rules, for example:
  - higher ROS lowers ATP and ETC activity
  - higher ROS increases γH2AX
  - higher cytosolic mtDNA increases inflammatory tone

### Were any real individuals involved?

No. There are no human participants, patient records, or experimental samples in this dataset.

## Benchmarking and source knowledge

### What informed the synthetic values?

The synthetic values were informed by peer-reviewed PubMed-indexed literature describing:

- hallmarks of ageing
- oxidative stress in ageing cells
- mitochondrial dysfunction
- cellular senescence
- IL-6-rich SASP biology
- proteostasis decline
- mtDNA-linked inflammatory signaling

### Are the values copied from published datasets?

No. Values were not copied directly from a single study. They were designed to be **directionally and approximately biologically plausible**, based on representative trends and ranges in the literature.

## Preprocessing and feature engineering

### Was any preprocessing done?

Yes. The dataset generation process included:

- clipping of implausible low values
- tissue-specific scaling
- embedded mechanistic couplings
- latent score construction

### How was the target created?

#### `repairability_score`
This continuous score was constructed as a weighted combination of variables expected to support or impair repair competence.

Positive contributors:
- ATP
- ETC activity
- proteostasis
- autophagy
- telomere length

Negative contributors:
- ROS
- γH2AX
- IL-6
- p16
- p21
- cytosolic mtDNA

#### `repair_window_open`
This binary label was generated by thresholding the continuous repairability score.

## Distribution

### How is the dataset distributed?

The dataset is generated within the Jupyter notebook rather than distributed as a fixed CSV in the current repository version.

### Why?

This allows complete transparency of the simulation logic and enables users to adjust parameters, noise levels, weights, and biological assumptions.

## Maintenance

### Who maintains the dataset?

The dataset is maintained as part of the Repairable Human Biology proof-of-concept repository.

### Will it be updated?

Potential future updates may include:

- exporting a static CSV version
- adding more tissues
- adding more explicit intervention variables
- integrating real public datasets
- generating multi-omics synthetic extensions

## Limitations

- fully synthetic
- not experimentally validated
- latent target is constructed, not measured
- simplified tissue abstraction
- simplified biological dependencies
- no true longitudinal sampling
- no batch effects or assay noise typical of real omics studies

## Biases and risks

The dataset reflects the assumptions embedded in its generation rules. Potential biases include:

- overemphasis on selected ageing hallmarks
- underrepresentation of tissue-specific biology
- simplified inflammatory and repair dynamics
- circularity between input features and derived targets

Users should therefore treat this dataset as a framework-building resource, not as a substitute for empirical biology.

## Recommended validations before real-world use

Before adapting this framework to real data, the following should be performed:

- validation in primary-cell or organoid datasets
- comparison with real senescence and ageing signatures
- testing against external cohorts
- tissue-specific recalibration
- uncertainty analysis
- causal perturbation experiments
